# Project Instructions

## Design System (applies globally)

All visual output produced in this project — web pages, artifacts, decks, graphics, posters, dashboards — must follow the Editorial Mono/Yellow design system defined in `design-system.md`.

Key rules (see `design-system.md` for the full spec):

- Black (`#0A0A0A`) editorial base, white text, one acid-yellow accent (`#D6FF3F`) used strictly for emphasis/signal — never as a background or photo tint
- Three type voices that never blend: huge condensed uppercase display, tiny letterspaced monospace meta-labels, quiet clean-sans body
- Asymmetric bento grid with hairline `#333333` dividers, no shadows or rounded corners
- Photography stays high-contrast monochrome; tool/brand icons are the only multi-color exception
- Numbered lists only for genuinely sequential content

## Voice

`soul.md` defines who the author is at a human level. Use it as the source of voice and story for any personal or brand copy.
