## Design System (applies globally)

All visual output produced in this project — web pages, artifacts, decks, carousels, graphics, infographics, dashboards, UI — must follow the Neon Grid design system defined in `design-system.md`.

Key rules (see `design-system.md` for the full spec):

- Near-black (`#0A0A0A`) base with a single neon-green accent (`#B8FF3C`) for headlines, borders, glows, icons — never a second bright accent
- Oversized condensed uppercase display type (Archivo Black/Anton style) as the loudest element, one per composition, next to small quiet body copy — nothing in between
- Content lives in isolated dark cards (`#111111`) with thin green or `#2A2A2A` borders; generous black space
- Faint graph/grid-paper texture on black at low opacity; soft green glow behind focal elements
- Photos duotone-treated toward black + green; portraits get circular crops with a thin green ring
- Tool/software icons keep their native brand colors — the one multi-color exception
