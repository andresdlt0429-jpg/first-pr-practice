# First PR Practice

A small repo for learning the GitHub pull request workflow.

## What this is

This repo exists to practice the full loop of making a change, pushing a
branch, and opening a pull request on GitHub.

## Getting started

1. Clone the repo: `git clone https://github.com/andresdlt0429-jpg/first-pr-practice.git`
2. Create a branch for your change: `git checkout -b my-change`
3. Make your edit, commit it, and push the branch
4. Open a pull request on GitHub
