# Design System — Neon Grid Portfolio

Extracted from the reference portfolio (Antonio Espana, Graphic Design 2017-2026). This is the visual language to apply to any carousel, graphic, slide, infographic, or UI I ask for.

## Color

| Token | Hex | Use |
|---|---|---|
| `--bg-black` | `#0A0A0A` | Base background, near-pure black |
| `--brand-green` | `#B8FF3C` | Primary accent — headlines, borders, glows, icons |
| `--brand-green-dim` | `#5C8A1E` | Muted green for secondary text/lines on black |
| `--white` | `#FFFFFF` | Primary text on black |
| `--gray-line` | `#2A2A2A` | Hairline dividers, card borders (non-accent) |
| `--card-black` | `#111111` | Card/panel fill, slightly lifted off base black |

Rule: black dominates, green is the single accent — it never competes with a second bright color. Software/tool icons keep their native brand colors as the one exception (Ps blue, Ai orange, Id pink).

## Typography

| Role | Face | Weight/Style | Notes |
|---|---|---|---|
| Display (hero words) | Heavy grotesque, e.g. Archivo Black / Anton | 900, uppercase | Set huge, tight letter-spacing, fills the width of its container. This is the loudest element on any layout — one per composition. |
| Headings | Same grotesque family, lighter | 700, uppercase | Section labels like "ABOUT ME," "EDUCATION," "SOFTWARES" |
| Body | Clean geometric sans, e.g. Inter / Helvetica Neue | 400–500 | Small, restrained, does the reading work the display type doesn't |
| Labels/eyebrows | Same as body | 500, uppercase, letterspaced | Small caps-style tags |

Contrast is the whole trick: enormous condensed display type next to small, quiet body copy. Nothing in between.

## Imagery Treatment

- Photos are duotone-treated toward black + green, or desaturated with a green-tinted glow/rim light
- Portraits: circular crop, thin green ring border, soft green glow behind the edge
- Hero/feature images can break out of their frame (e.g., statue overlapping the headline type)

## Texture & Background

- Faint graph/grid paper pattern overlaid on black at low opacity — gives depth without adding color
- Large geometric shapes (triangles/arrows) in flat green or green-outline, used as background accents, not focal content
- Soft green glow/blur behind key focal elements (portrait, statue) simulating neon backlight

## Layout Patterns

- Generous black space; content sits in isolated dark cards (`--card-black`) with either a solid or 1px green border
- Cards are self-contained info blocks (About, Education, Contact, Software) — no bleeding content across cards
- Icon badges: small rounded-square tiles, one tool logo each, native brand colors, arranged in a tight row/grid
- Section labels sit top-right or as a small tag, never competing with the display headline for attention

## Signature Element

The combination that makes this unmistakably this brand: **black + single neon-green accent + grid-paper texture + oversized condensed display type + duotone glow photography.** Any one of these alone is generic; together they're the identity. Don't add a second accent color or a second display face — the restraint is the point.

## Do / Don't

| Do | Don't |
|---|---|
| One accent color (green) | Add a second bright accent |
| Oversized display type as the loudest element | Let body copy compete in size |
| Keep cards dark with thin green edges | Use white card backgrounds |
| Let icons keep native brand colors | Recolor tool icons green |
| Use the grid texture subtly | Let texture overpower content |
