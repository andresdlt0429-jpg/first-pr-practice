# Design System — Editorial Mono/Yellow

Extracted from the reference portfolio (Mariana Napolitani, brand/design portfolio). This is the visual language to apply to any page, deck, or graphic I ask for.

## Color

| Token | Hex | Use |
|---|---|---|
| `--bg-black` | `#0A0A0A` | Base background |
| `--accent-yellow` | `#D6FF3F` | The single accent — name, underlines, highlighter marks, numbers, bar charts, QR panel fill |
| `--white` | `#FFFFFF` | Primary text, headlines |
| `--gray-panel` | `#1A1A1A` | Card/panel fill, one step off black |
| `--gray-line` | `#333333` | Hairline dividers between modules |

Rule: pure black-and-white editorial base, one acid-yellow accent doing every job that needs emphasis — name, highlighter strokes over key words, numbering, data bars, and the single warm CTA panel (QR code block). No second accent color anywhere.

## Typography

| Role | Face | Weight/Style | Notes |
|---|---|---|---|
| Display (hero word) | Heavy grotesque/condensed sans | 900, uppercase | "PORTFOLIO" — fills the full width, the loudest single element on the page |
| Meta/nav labels | Monospace | 400–500, uppercase, letterspaced | Tiny top-of-page labels like section tags, date, barcode-adjacent text — reads as editorial/technical annotation |
| Name/eyebrow | Same grotesque as display, smaller | 800, uppercase | Set in the accent yellow |
| Body copy | Clean sans | 400 | Short paragraphs, tight line length, quiet next to the display type |
| Pull-quote | Bold sans, mixed with a highlighter treatment | 700–800 | One key phrase per quote gets a yellow highlighter-marker underline/box |
| Numerals | Grotesque, bold | 700 | Used for a genuine numbered list (services 01–05, work items) in yellow |

The type system has three distinct voices — huge display, tiny mono meta-text, and readable body — and they never blend into each other.

## Imagery Treatment

- Portrait photography stays high-contrast black and white / near-monochrome, sunglasses or shadow often obscuring part of the face for an editorial, slightly anonymous feel
- Work-sample thumbnails (branding, web, editorial, packaging) are shown dark and slightly desaturated inside their cards, letting the yellow card labels do the identifying work
- No warm skin tones or bright photography — everything is pulled toward black/white/yellow

## Background & Texture

- Flat black, no gradient, no grain — the editorial feel comes from typography and layout, not surface texture
- Swiss/editorial meta-marks (barcode graphic, small asterisks, tiny corner icons) scattered as design furniture, not illustration
- Highlighter-marker yellow strokes behind key words in quotes are the only "hand-made" texture in the system

## Layout Patterns

- Asymmetric bento/module grid: uneven card sizes packed edge-to-edge with thin gray dividers, no card shadows or rounding
- A numbered list module (services, education, or process) is used for genuinely sequential content only
- Bar-chart style skill/expertise indicators: horizontal yellow bars of varying length against black, minimal labeling
- Tool/software icons shown as small square badges in a tight row, native brand colors allowed as the one multi-color exception
- Footer repeats the display-type treatment for a closing CTA ("LET'S CREATE") paired with a small contact block and a yellow QR panel

## Signature Element

The combination that makes this unmistakably this system: **black-and-white editorial base + a single acid-yellow accent used for every emphasis job (name, highlighter marks, numbers, charts, CTA panel) + huge condensed display type + tiny monospace meta-labels borrowed from print/Swiss design.** The yellow never spreads into imagery or backgrounds — it's reserved strictly for signal, not decoration.

## Do / Don't

| Do | Don't |
|---|---|
| Use yellow only for emphasis/signal elements | Use yellow as a background or photo tint |
| Keep meta-labels in monospace, small and quiet | Let meta-labels compete with the display headline |
| Use numbered lists only for real sequences | Add numbering as pure decoration |
| Keep photography monochrome/high-contrast | Introduce warm, saturated photography |
| Let tool icons keep native brand colors | Force tool icons into the yellow/black palette |
